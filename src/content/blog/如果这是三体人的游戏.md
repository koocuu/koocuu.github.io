之前在越南和朋友旅行的时候遇到了一些很不顺的事情，朋友调侃到：估计是你背后的三体人拿n+1了，没给你的账号充钱。当时会心一笑，但后来细想，如果我们的人生的确就是三体人的游戏呢？

# 1.小学
有清晰的记忆的起点是从中梁山搬到玉清寺的出租房，转校到人和小学的四年级。不知道是不是思考的东西很少，只想着玩游戏和学武术，那个时候我并不觉得时间过得很快；相反，我觉得我身后有无穷无尽的时光，无穷的时光在空调房，吃着西瓜，玩着游戏，今天是这样，明天也是这样，以后都是这样。

# 2.初中
又要讲那个说了很多遍的故事了：初中我的英语成绩只有47.5分(满分150)。我的父母对我非常失望，他们认为我的人生就这样了，以后只能当一个没有出息的街溜子。后来我听Avril的歌，看英语电影，入门了英语，后面成绩越来越好；然后我去了一所还不错的高中。回忆中有很多细节清晰可见，但好像也是一段完整经历。

# 3.高中
我去了一所还不错的高中。Study like a nerd，当然我也经常偷懒，因为我是一个比较懒的人。高考我正常发挥，在选择大学的时候，我的父母叫我去警校，我不想去，但我最终屈服了，这也导致我没有选择俄语专业（警校和俄语都是提前批，只能二选一）。后面警校没有录取我，我去了一所还不错的大学学习计算机(如果我当时选择了俄语，我就肯定会被俄语录取，也就不会学习计算机了)。虽然记忆里的高中充满了数学公式和古诗词，单休日的守望先锋和晚自习的小说，但总的来说也算充实，并没有觉得这3年无声溜走。

# 4.大学
大学我经历了挂科、重修、挫败、躺平，还经常感觉自己是个失败者。但最终我获得了美团、字节、网易的实习offer，去了上海美团实习并且我转正了。一方面我还挺幸运，这是个不错的地方，对于一个小镇做题家来说，我已经很幸运了，我有了选择自己命运的权利。虽然我从来没想过我的未来会在上海展开。大学的记忆和情绪特别清晰，过得快吗？快，但是感觉和之前的时光也没有太大的割裂感。

# 5.工作后
毕业后去了美团开启了自己的第一份正式工作，良心来说美团作为第一份工作薪资和平台都很不错。然后今年5月到爱奇艺开启了自己的第二份工作。上学的时候有寒暑假，期末考；但工作后好像很久没有大的记忆锚点了。也许就正是在这一周一周重复的工作和生活里，时间变得更快了。

# 6.结语
这篇没有主线的文章，不知不觉就发散到了我的过去，发散到了一些怀念的时光里。
在我怀念的时光里，很多是发生之前无比抗拒，甚至会觉得无趣的，例如一次“漫无目的”的学校闲逛、一次会觉得疲惫的千岛湖团建。但是过了之后，又会觉得，那一刻我好像永远失去了。
小学时候就在背诵但从未理解的“此情可待成追忆,只是当时已惘然”在此刻仿佛有了具象化的含义。
